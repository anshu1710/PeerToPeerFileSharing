Specifications-

DC file (which also uses ClientServer file) has to be run on client side.
Hub file( which also uses Connector for database connectivity ) has to
be run on Server side.

File->New (on run of DC) is used of uplaoding the folder specified
which result in uploading of all files of that folder.
File->Query send the Text written in text area as Query(filename) to hub.
File->Exit tells the hub that client is leaving.
File->Path to specify the path at which downloaded file will be saved. (by deafult file is saved at program location).
